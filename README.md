# Раскладки

## MacOS

Это кастомная раскладки для macOS.

### Особенности

- Стандартные `QWERTY` и `ЙЦУКЕН`
- Нету буквы `ё` и `Caps Lock` за ненадобностью
- Знаки препинания и спецсимволы вынесены в единый `Option`-слой и полностью совпадают в обеих раскладках
- `Caps Lock` используется как тот самый `Option`-слоификатор
- Через программу Karabiner добавляются следующие шорткаты:
  - `Option + 1` → English
  - `Option + 2` → Russian
  - `Option + W/A/S/D` → `Arrows`
  - `Option + Q/E` → `Home / End`
  - `Option + Z/C` → `Page Up / Page Down`
  - `Option + R` → `Backspace`
  - `Option + X` → `Delete`

### Установка

1. Кладём `.bundle` в папку `~/Library/Keyboard Layouts`
2. Перезагружаем Mac
3. Открываем `System Settings → Keyboard → Input Sources`
4. Добавляем раскладки:
   - `English (C)`
   - `Russian (C)`
5. Скачиваем и устанавливаем [Karabiner-Elements](https://karabiner-elements.pqrs.org)
6. В `Karabiner-Elements → Complex Modifications` нажимаем `Add your own rule`
7. Вставляем туда содержимое `karabiner-caps-script.json` и сохраняем

### Пояснения

Если начать переставлять буквы, цена переучивания резко вырастает: ломается слепая печать, снижается скорость, становится неудобно печатать на чужих устройствах и сложнее вернуться к обычной клавиатуре.

`Option`-слой позволяет вынести знаки препинания и клавиши для редактирования текста, которые обычно находятся очень далеко, из-за чего редко используются.

`Caps Lock` — это бесполезная клавиша, которая занимает одно из самых удобных мест: сразу слева от левого мизинца. Поэтому логично превратить её в слоефикатор.

Буква `ё` используется сравнительно редко. Для повседневного набора большинству достаточно буквы `е`. Это сознательный компромисс: не занимается ценный слот в базовом слое.

### Почему знаки препинания в слое не как на стандартных клавиатурах

Слой — это уже не исторически сложившаяся раскладка, а полностью управляемая зона. Его цель не “копировать стандарт”, а быть **удобным**.

Поэтому в слое знаки можно располагать так, как выгодно для реальной печати:
- рядом по смыслу
- рядом по частоте использования
- так, чтобы было удобно нажимать вслепую
- так, чтобы одинаково работало и в русском, и в английском

### Почему строгое переключение языков, а не цикличное

У циклического переключения есть фундаментальная проблема: чтобы выбрать нужный язык, нужно помнить, какой активен сейчас, и иногда нажимать переключение несколько раз. При строгом переключении, если вы сомневаетесь, какой сейчас язык, то вы просто переключаетесь на нужный язык на всякий случай и сразу начинаете писать.

Вторая ключевая причина заключается в том, что в macOS нельзя избавиться от системной английской раскладки. В итогу у вас будет три раскладки: системная и кастомная английские и кастомная русская. Переключать их напрямую гораздо удобнее и быстрее, чем циклично проклацивать стандартным способом.

## Windows: Moonlander

Это моя раскладка для клавиатуры Moonlander.

Вследствие того, что я не умею кодить на C, мне пришлось изменять системные раскладки с помощью [MSKLC](https://www.microsoft.com/en-us/download/details.aspx?id=102134).

### Установка для Windows

Скачиваем [мою прошивку](https://configure.zsa.io/moonlander/layouts/WazEM/latest/0) и [программу для прошивки](https://www.zsa.io/flash), после чего прошиваем клавиатуру.

Скачиваем этот репозиторий и устанавливаем модифицированную Русскую раскладку с `setup.exe`. Старую русскую раскладку можно удалить. Перезагружаем систему.
