# Раскладки

## macOS

Это кастомные раскладки для macOS.

### Особенности

- Базовые слои остаются стандартными: `QWERTY` и `ЙЦУКЕН`
- В базовом слое нет `ё` и `Caps Lock`
- Знаки препинания и спецсимволы вынесены в единый `Option`-слой, одинаковый для русского и английского
- `Caps Lock` превращён в `Option`-модификатор, чтобы слой был прямо под левым мизинцем
- Правый `Command` переключает языки по кругу между `English (C)` и `Russian (C)`
- Если активна не одна из кастомных раскладок, первое нажатие правого `Command` возвращает в `English (C)`
- Через Karabiner добавляются следующие шорткаты:
  - `Caps + W/A/S/D` → `Arrows`
  - `Caps + Q/E` → `Home / End`
  - `Caps + Z/C` → `Page Up / Page Down`
  - `Caps + R` → `Backspace`
  - `Caps + X` → `Delete`
- Физические стрелки справа снизу отключаются скриптом, чтобы не было соблазна тянуться к ним по привычке и переучивание на слой стрелок под `Caps` шло быстрее

### Установка

1. Кладём `.bundle` в папку `~/Library/Keyboard Layouts`
2. Перезагружаем Mac
3. Открываем `System Settings → Keyboard → Input Sources`
4. Добавляем раскладки:
   - `English (C)`
   - `Russian (C)`
5. Скачиваем и устанавливаем [Karabiner-Elements](https://karabiner-elements.pqrs.org)
6. В `Karabiner-Elements → Complex Modifications` нажимаем `Add your own rule`
7. Вставляем туда содержимое `karabiner-caps-script.json` и сохраняем

### Пояснения

Если начать переставлять буквы, цена переучивания резко вырастает: ломается слепая печать, падает скорость, становится неудобно печатать на чужих устройствах и сложнее вернуться к обычной клавиатуре.

`Option`-слой позволяет вынести знаки препинания и клавиши навигации туда, где они действительно доступны вслепую. Вместо дальних углов клавиатуры получаем слой под `Caps`, на котором живут стрелки, `Home / End`, `Page Up / Page Down`, `Backspace` и `Delete`.

`Caps Lock` — одна из самых удобных по расположению клавиш, но сама по себе почти бесполезна. Поэтому логичнее использовать её как модификатор слоя.

Буква `ё` используется сравнительно редко. Для повседневного набора большинству хватает `е`, поэтому отдельный слот в базовом слое не тратится.

### Почему знаки препинания в слое не как на стандартных клавиатурах

Слой — это уже не исторически сложившаяся раскладка, а полностью управляемая зона. Его цель не копировать стандарт, а быть удобным.

Поэтому знаки в слое имеет смысл раскладывать так, как выгодно для реальной печати:
- рядом по смыслу
- рядом по частоте использования
- так, чтобы их было удобно нажимать вслепую
- так, чтобы они одинаково работали и в русском, и в английском

### Почему переключение языков на правом `Command`

Правый `Command` работает как простой переключатель по кругу между двумя нужными раскладками: `English (C)` и `Russian (C)`. Это одно короткое действие, которое легко нажимать большим пальцем правой руки, не отрываясь от основной позиции.

При этом системная английская раскладка не попадает в этот круг. Если вы по какой-то причине оказались не в кастомной раскладке, первое нажатие правого `Command` возвращает вас в `English (C)`, а дальше переключение снова идёт только между двумя основными языками.

### Почему отключены физические стрелки

Если оставить рабочими стрелки справа снизу, рука почти всегда будет тянуться именно к ним. Из-за этого слой стрелок под `Caps` осваивается медленнее или не осваивается вообще.

Поэтому скрипт отключает физические стрелки и оставляет рабочим только слой `Caps + W/A/S/D`. Это небольшое принуждение, которое заметно ускоряет переучивание.

## Windows: Moonlander

Вдохновение:
- [статья от @optozorax](https://optozorax.github.io/p/my-keyboard-layout)
- https://dreymar.colemak.org/ergo-mods.html

Это моя раскладка для клавиатуры Moonlander.

Поскольку я не умею писать на C, пришлось менять системные раскладки с помощью [MSKLC](https://www.microsoft.com/en-us/download/details.aspx?id=102134).

### Установка для Windows

Скачиваем [мою прошивку](https://configure.zsa.io/moonlander/layouts/WazEM/latest/0) и [программу для прошивки](https://www.zsa.io/flash), после чего прошиваем клавиатуру.

Скачиваем этот репозиторий и устанавливаем модифицированную русскую раскладку через `setup.exe`. Старую русскую раскладку можно удалить. После установки лучше перезагрузить систему.
